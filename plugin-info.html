Plugin Documentation

Goals available for this plugin:

Goal Description
inject:help Display help information on inject-maven-plugin.
Call mvn inject:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
inject:inject Injects properties into compiled Java code, as constant or method return value.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.6
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>de.m3y.maven</groupId>
          <artifactId>inject-maven-plugin</artifactId>
          <version>1.2-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>de.m3y.maven</groupId>
        <artifactId>inject-maven-plugin</artifactId>
        <version>1.2-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
